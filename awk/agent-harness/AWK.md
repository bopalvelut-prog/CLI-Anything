# AWK CLI-Anything Harness

**Binary**: `awk`  
**Description**: Text processing

## Quick Start
```bash
python -m cli_anything.awk repl
```

## Architecture
- `cli_anything/awk/core/session.py` — Project session management
- `cli_anything/awk/core/` — awk-specific operations
- `cli_anything/awk/utils/repl_skin.py` — Terminal UI
- `cli_anything/awk/skills/SKILL.md` — Skill definition
