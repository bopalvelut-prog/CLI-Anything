# FIND CLI-Anything Harness

**Binary**: `find`  
**Description**: File search

## Quick Start
```bash
python -m cli_anything.find repl
```

## Architecture
- `cli_anything/find/core/session.py` — Project session management
- `cli_anything/find/core/` — find-specific operations
- `cli_anything/find/utils/repl_skin.py` — Terminal UI
- `cli_anything/find/skills/SKILL.md` — Skill definition
